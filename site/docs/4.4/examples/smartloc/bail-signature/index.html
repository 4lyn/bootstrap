Comment ça marche ?

Chaque signataire a reçu un email d'invitation pour signer le bail. Si un des signataires n'a toujours pas signé le bail en ligne, nous vous conseillons de le contacter pour voir si l'email n'est pas dans ses spams ou s'il n'a pas eu un problème à la signature.

Vous pouvez modifier les coordonnées des signataires en cas d'erreur en cliquant sur "modifier" en face de chaque signataire.

Il n'est par contre pas possible de modifier le fichier qui a été envoyé en signature si vous vous êtes trompé sur un élément du bail. Le cas échéant, cliquez sur "annuler la signature", il faudra alors modifier le bail, réactiver une option de signature et relancer les signatures. Vous pourrez joindre notre équipe support pour vous faire rembourser le deuxième paiement.

Bail signé !

Le bail signé vous a été envoyé par mail. N'oubliez pas de récupérer le fichier PDF signé depuis le mail qui vous a été envoyé et de le sauvegarder précieusement avant de clore la signature.

Êtes-vous sûr de vouloir annuler la signature ?

Si vous souhaitez simplement modifier les coordonnées de signataires, cliquez sur "Modifier" en face du signataire, puis cliquer sur "Relancer les invitations".

Si vous avez fait une erreur sur le bail, vous pouvez effectivement annuler. Il faudra alors modifier le bail, réactiver une option de signature et relancer les signatures. Vous pourrez joindre notre équipe support pour vous faire rembourser le deuxième paiement.
