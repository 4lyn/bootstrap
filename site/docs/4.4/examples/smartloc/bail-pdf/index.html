Comment fonctionne la signature électronique ?

En pratique voici les étapes :

  1. Vous générez votre bail à partir de cette page
  2. Vous consultez votre bail généré automatiquement au format pdf et le faites relire aux locataires
  3. Vous renseignez les coordonnées mail et téléphone de tous les signataires qui recoivent un mail d'invitation à signer
  4. Chaque partie signe avec un code SMS, le document est alors verrouillé par un certificat cryptographique
  5. Une fois que toutes les parties ont signé, vous recevez un mail de confirmation avec le document définitif; si un des locataires tarde à signer, vous disposez d'une page de suivi pour le relancer

Quels sont les avantages de la signature électronique ?

Satisfait ou remboursé Seulement 4€ / signataire

La signature électronique vous permet de signer rapidement avec une date d'effet dans le futur, et d'éviter ainsi que le locataire se désiste.
