Aide sur les justificatifs de revenus complémentaires

1. Pour les APL il est important de transmettre une simulation sur le logement futur et non une attestation passée.

2. Pour les revenus locatifs il est nécessaire de transmettre le bail et un document prouvant la propriété de chaque bien constituant le revenu foncier.
